# pool-deck-plans

**Version: v11.1 — Picture-Frame Double Border**

Two-section freestanding pool deck, 22' total width × ~18' depth.
- Two 11' × 11.5' sections with 6" center seam
- Double-row picture-frame border on all sides (11" total depth)
- Inner field dimensions: 9' 2" × 9' 6" per section
- 2×6 joists front-to-back at 16" O.C., full 11.5' length
- Border blocking every 16" O.C. under all double-row borders
- 5/4×6 decking with 1/8"–3/16" gaps

**Files:**
- `build_guide.html` — Full construction guide
- `framing_plan.html` — Cut list and framing details
- `quick_reference.html` — Quick specs and checklist
- `stake_string.html` — Site layout and staking guide
- `pool_deck_plan.md` — Design overview (markdown)
- `diagrams/joist_layout_v11.1.png` — Joist layout
- `diagrams/decking_layout_v11.1.png` — Decking layout
